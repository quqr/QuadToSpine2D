# QuadToSpine2D

QuadToSpine2D 用于将 QUAD 格式的 2D 动画数据转换为 Spine 2D 格式，同时也支持动画实时预览。

## 🔧 安装与使用

### 1. 使用流程

#### 转换器页面
1. **选择 QUAD 文件**：点击 "Open Quad File" 按钮选择你的 .quad 文件
2. **添加图像资源**：
   - 点击 "+" 按钮添加新的图像组
   - 为每个组添加对应的图像文件路径
   - 确保图像顺序正确
3. **开始转换**：点击 "Process Data" 按钮开始转换过程
4. **获取结果**：转换完成后，点击生成的链接获取 Result.json 文件

#### 预览器页面
1. **加载资源**：
   - 选择 QUAD 文件
   - 添加图像文件路径
   - 点击 "Load" 加载数据
2. **动画控制**：
   - 使用播放、暂停、帧控制等功能

## ⚙️ 配置选项

### 转换器设置
- **缩放因子**：调整输出图像的缩放比例
- **循环动画**：设置是否启用动画循环
- **JSON 格式化**：选择输出 JSON 的格式化方式
- **保存路径**：自定义结果文件和图像的保存位置

### 预览器设置
- **画布大小**：调整预览画布的尺寸
- **帧率控制**：设置动画播放的 FPS
- **图像缩放**：控制预览图像的显示比例
## ⚠️ 已知问题

1. **动画顺序问题**：某些复杂动画可能显示顺序不正确
2. **图层显示缺失**：部分动画图层可能无法正常显示
3. **文件兼容性**：某些特殊的 QUAD 文件格式可能无法转换

*如需获取 QUAD 文件，请参考 [psxtools 文档](https://github.com/rufaswan/Web2D_Games/blob/master/docs/psxtools-steps.adoc)*
